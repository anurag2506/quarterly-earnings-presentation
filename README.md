# Q4 2024 Quarterly Earnings Presentation

A professional RevealJS presentation for quarterly earnings reporting with advanced features.

## Features

✅ **Email Contact**: 24f3002629@ds.study.iitm.ac.in  
✅ **Markdown Slides**: Full markdown support for content  
✅ **Animated Fragments**: Progressive reveal of bullet points  
✅ **Syntax Highlighting**: Python and SQL code examples  
✅ **Mathematical Equations**: Financial formulas using LaTeX  
✅ **Speaker Notes**: Detailed guidance for each slide  

## Slide Contents

1. **Title Slide** - Introduction with email
2. **Executive Summary** - Written in Markdown with key metrics
3. **Financial Performance** - Animated metric cards
4. **Revenue Formula** - Mathematical equations
5. **Profitability Analysis** - More financial formulas
6. **Python Code** - Data analysis pipeline
7. **SQL Code** - Database query example
8. **Growth Trends** - Animated progression
9. **Investment Analysis** - Complex mathematical formulas
10. **Contact & Next Steps** - Call to action
11. **Closing Slide** - Thank you

## Controls

- **Arrow Keys**: Navigate between slides
- **Space**: Advance slide/fragment
- **S**: Open speaker notes in new window
- **ESC**: Show all slides overview
- **F**: Fullscreen mode
